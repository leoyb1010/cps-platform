# Commercialization Notes

## Sellable Position

Agent Studio is an agent-first social media operations system for creators, studios, and small teams that want local browser execution rather than handing account passwords to a SaaS.

The wedge:

- Agent researches platforms, competitors, comments, and web sources.
- Agent generates grounded topic candidates and platform-native content packs.
- HTML/CSS renderer creates stable carousel and long-image assets.
- Codex app uses the user's logged-in Mac browser to publish, schedule, maintain comments, and read analytics.
- Results become brand learnings for the next cycle.

## Why It Is Different

Most tools stop at "write a post" or "schedule a post". Agent Studio owns the operating loop:

```text
collect -> brief -> topic -> pack -> asset -> codex browser task -> publish -> comments -> analytics -> brand learning
```

The hard parts are:

- source-grounded research
- platform-native adaptation
- local authenticated browser execution
- traceable publishing
- comment-to-topic feedback
- analytics-to-learning memory
- account safety boundaries

## Commercial Requirements

- workspace and brand voice profile
- per-account posting limits
- task and browser trace retention
- role-based approvals for Autopilot Mode
- exportable source, task, comment, and analytics data
- private model/provider option
- clear forbidden actions policy

## Seed Users

- AI tool creators running multi-platform accounts
- career consultants and education creators
- boutique agencies serving founder-led brands
- overseas SaaS founders repurposing technical content
- small MCNs that need local authenticated execution for Chinese platforms

## Product Modes

- Draft Mode: Agent fills content and stops at confirmation.
- Assist Mode: Agent prepares everything and waits for user approval.
- Autopilot Mode: Agent can publish, schedule, and reply within explicit rules, limits, and trace requirements.

Autopilot is a paid/enterprise capability because it requires policy, logs, account pacing, and review controls.
