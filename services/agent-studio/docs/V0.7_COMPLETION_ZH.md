## 已交付能力

### 1. Template Registry（模板注册表）

- 数据：`registry/recipes.json`（12 条 manifest）
- 逻辑：`src/lib/templateRegistry.js`（推荐 / 加权随机 / 手动指定）
- 校验：`npm run registry:validate`

### 2. 模板选择 UX

- **Visual Studio** 内嵌 `TemplateGallery`：智能推荐、随机探索、手动指定、再随机一款
- **排除 AGPL** 开关（默认开启，歸藏系不进随机池）
- 偏好持久化：`localStorage` → `content-os-template-prefs`
- Preview Hub 显示当前生效模板名

### 3. 服务端 API

- `GET /api/templates/registry` — 按平台过滤列表
- `POST /api/templates/pick` — 解析将使用的 recipe + visualStyle
- `POST /api/templates/report` — 导出成功/失败权重反馈（SQLite `template_weights`）
- `POST /api/templates/lint` — 导出前 HTML 规则检查

导出链路（cover / xhs-carousel / export-png / render-html）已接入 `pickRecipe` + `resolveVisualStyleFromRecipe`。

### 4. 质量闸门

- `server/src/visualLinter.js`：字号、占位文案、空分镜、CJK 字体栈
- cover 导出失败返回 **422** + `lint` 详情
- `src/lib/fontRegistry.js`：Noto Sans/Serif SC + 1080 字阶 token，已注入 XHS carousel HTML

### 5. 真机框 v2

- `DeviceFrame`：Dynamic Island 缺口 + 状态栏 + Home 条
- Preview Hub 三列：小红书 / 抖音 / X
